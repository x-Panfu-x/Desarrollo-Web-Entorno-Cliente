# 💻 Desarrollo Web en Entorno Cliente

Repositorio dedicado a las prácticas, ejercicios y proyectos de la asignatura **Desarrollo Web en Entorno Cliente (DWC)**.

---

## 📂 Organización del código

El contenido del curso está dividido por trimestres:
*   📁 **1st Trimestre:** Primeros pasos, fundamentos de JavaScript, manipulación del DOM y eventos.
*   📁 **2nd Trimestre:** Desarrollo avanzado, asincronía (promesas, fetch/APIs) y uso de frameworks o librerías del entorno cliente.

---

## 🛠️ Tecnologías utilizadas
*   HTML5 / CSS3
*   JavaScript (ES6+)
